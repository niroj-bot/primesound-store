# PrimeSound Store

A Java-based web application for an online shopping system that allows users to browse products, manage a cart, and simulate a purchase flow.

---

## 📌 Overview

PrimeSound Store is a web application developed using Java (JSP/Servlet) as part of my academic project.

The system provides basic e-commerce functionality including product browsing, cart management, and user authentication. Through this project, I learned how a typical online shopping system is structured and how different components interact.

## 🚀 Features

- User authentication (Login / Logout)
- Product listing and detail view  
- Add items to cart  
- View and manage cart items  
- Basic purchase processing  
- User registration and profile handling  
- Admin-side item management (add/update/delete items)  

---

## 🧩 System Flow

1. User logs in or registers  
2. Browse product list  
3. View item details  
4. Add items to cart  
5. Review cart items  
6. Complete purchase (sale processing)
